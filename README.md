# Simple Interest Calculator

This project is a Simple Interest Calculator that calculates the simple interest based on user-provided inputs.

## Description
The calculator takes the following inputs:
- Principal amount
- Rate of interest
- Time period

It then computes the simple interest using the standard formula.

## Formula
Simple Interest (SI) is calculated as:

SI = (Principal × Rate × Time) / 100
